

Daily Schedule

A simple interactive daily schedule webpage that shows timers, progress bars, and highlights the current task automatically based on your system time.

Features
	•	Real-time active task detection
	•	Countdown timers for each session
	•	Morning / Afternoon / Evening tabs
	•	Expand / collapse task details
	•	Progress bars that fill automatically
	•	“Jump to Now” button
	•	Works fully offline (just open the file)

How to Use
	1.	Open the website:
	https://nishaaaaal.github.io/daily_schedule/

	2.	Or open the file locally by double-clicking index.html.

File Info
	•	index.html — Main schedule interface

Purpose

A personal project to help structure and track my daily routine visually.
